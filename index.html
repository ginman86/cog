Cog

cog is a cognitive psychology evaluator developed for use in the labs @ Oklahoma University.

Download this project as a .zip file Download this project as a tar.gz file

The main goal of cog is to provide a way to administer studies that is accessible, configurable and scalable.

Programs

cog is designed to run off of programs. A program is a collection of question block/mode pairs. Programs can be setup by the administrator to change the content of a study at any time.

day1:
[
   {
      mode:  modes.study,
      block: 2
   },
   {
      mode:  modes.test,
      block: 2
   }
]

Local deployment

cog is designed to be deployed locally with no communication to a server. This is ideal when dealing with the constraints of a scientific study. It is 100% self sufficient and interfaces with a local proxy that can easily be swapped out with server proxy if desired.
